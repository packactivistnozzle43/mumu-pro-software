### 💡 实战技巧

**技巧 1：调整 CPU 和内存分配**  
打开 MuMu 多开器 → 点击设置 → 性能设置 → 将 CPU 和内存调整为你电脑配置的一半。不要拉满，否则会影响系统运行。

**技巧 2：开启「极速模式」**  
右上角菜单 → 开启「极速模式」。这个选项会优先把资源给前台游戏，后台多开的其他模拟器自动降低资源占用。

**技巧 3：设置「FPS 上限」**  
如果你的显示器是 60Hz，把游戏 FPS 锁在 60。如果显示器是 144Hz，最高可以开到 120 FPS。超过显示器刷新率的帧数只是增加硬件负担，肉眼无感。

**技巧 4：导入云端按键配置**  
热门游戏（吃鸡、LOL 手游、CODM）都有大神分享的「高分段键位配置」。在键位设置里搜索「排行榜」，一键导入，不用自己慢慢调。

### ❌ 游戏闪退 / 黑屏

**原因：** 渲染模式不兼容或显卡驱动过旧。

**解决方案：**
1. 在 MuMu Pro 设置中切换渲染模式（DirectX 12 / Vulkan / OpenGL）
2. 更新显卡驱动到最新版本
3. 为该游戏分配更多内存（4GB → 8GB）

### ❌ 卡顿 / 掉帧

**原因：** CPU/内存分配不足，或电脑电源模式为「省电」。

**解决方案：**
1. Windows 设置 → 电源选项 → 开启「高性能」模式
2. MuMu 设置 → 性能 → CPU 核心拉满，内存拉到 8GB
3. 开启「极速模式」

### ❌ 键鼠映射不生效

**解决方案：**
1. 确认游戏内操作模式为「摇杆」模式
2. 在 MuMu Pro 侧边栏点击「键位设置」→「重置默认」
3. 重新启动游戏
